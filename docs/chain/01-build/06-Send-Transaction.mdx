## Send Transaction

Sending a transaction on Push Chain is pretty straight forward and consists of 2 steps:

- Calling the `createUnsigned` to create an Unsigned Transaction
- Send transaction

To send the transaction a Signer is necessary. In the example below we will be importing a private key from the `.env` and using the `Account` class from `viem` library to sign the transaction, but you can use any other library.

<Tabs className="codetabs" groupId="code-examples">
  <TabItem value="js" attributes={{className: "codetab js"}} default>
    First, let's initialize our viem Account. This will be used for signing when sending a transaction

    ```typescript
    const senderEthereumPrivateKey = process.env.ETHEREUM_PRIVATE_KEY;
    const ethereumSenderAccount = privateKeyToAccount(senderEthereumPrivateKey as `0x${string}`);
    ```

    In this example we will be sending a transaction with category `MY_EMAIL_APP` and the following data:

    ```typescript
    const email = {
      title: 'Hello old friend from Solana!',
      message: 'Greetings from Ethereum world.'
    };
    ```

    Now, let's create our signer object to sign messages that are sent to Push Chain:

    ```typescript
    const signer = {
      account: `eip155:1:${ethereumSenderAccount.address}`,
      signMessage: async (data: Uint8Array): Promise<Uint8Array> => {
        return hexToBytes(await ethereumSenderAccount.signMessage({message: {raw: data}}));
      }
    };
    ```

    Let's serialize the `email` object:

    ```typescript
    const serializedData = new TextEncoder().encode(JSON.stringify(data));
    ```

    Now, let's create our Unsigned Transaction. We will be sending this transaction to the Ethereum mainnet address `0x8F6325974108C7041B8035131eCe8A89708d02d0` and add as `category` for this example: `MY_EMAIL_APP`:

    ```typescript
    const unsignedTransaction = pushNetwork.tx.createUnsigned(
      'MY_EMAIL_APP',
      ['eip155:1:0x8F6325974108C7041B8035131eCe8A89708d02d0'],
      serializedData
    );
    ```

    Let's send our transaction:

    ```typescript
    const transactionHash = await pushNetwork.tx.send(unsignedTransaction, signer);
    ```

</Tabs>

### Create Unsigned Transaction parameters

| Param       | Type                    |  Remarks                                                                                                                                                                        |
| ----------- | ----------------------- |  ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `category`| `string`                |  Transaction Category                                                                                                                                   |
| `recipients` | `string[]`         | Recipients of the transaction                                                                                                                                                        |
| `data`           | `Uint8Array` |  Array of Bytes for the transaction data                                                                                                                                                    |

### Send Unsigned Transaction parameters

| Param       | Type                    |  Remarks                                                                                                                                                                        |
| ----------- | ----------------------- |  ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `unsignedTx`| `Transaction` |  Unsigned Transaction that was created above by calling the `createUnsigned` method |
| `signer` | `Signer` | Object to sign the transaction that will be submited to Push Chain |

Where `Signer` is defined as:

```typescript
{
  account: string;
  signMessage: (dataToBeSigned: Uint8Array) => Promise<Uint8Array>;
}
```

| Param       | Type                    |  Remarks                                                                                                                                                                        |
| ----------- | ----------------------- |  ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `account`| `string` |  Address in CAIP-10 format with the address that will be used to sign the Transaction |
| `signMessage` | `(dataToBeSigned: Uint8Array) => Promise<Uint8Array>` | Function to sign an array of bytes |
