## Send Transaction

Sending a transaction on Push Chain is straightforward. Below is an example of how to send a transaction using the Push Chain SDK.

```typescript
send: (
  recipients: UniversalAccount[],
  options: {
    category: string;
    data: Uint8Array;
  }
) => Promise<{ txHash: string }>;
```

| **Param**     | **Type**                | **Remarks**                                                                                       |
|---------------|-------------------------|---------------------------------------------------------------------------------------------------|
| `recipients`  | `UniversalAccount[]`   | An array of recipient addresses in a chain-agnostic format. Each address specifies the destination for the transaction. |
| `category`    | `string`               | The category of the transaction, used to classify or tag the transaction (e.g., `example-category`). |
| `data`        | `Uint8Array`           | Serialized data payload for the transaction.                                                      |

### Initialize the Universal Signer
Generate a private key and account, then create a Universal Signer object. We will use `viem` to generate the private key and account.

```typescript
import { PushChain, CONSTANTS } from '@pushprotocol/push-chain';
import { hexToBytes } from 'viem';
import { privateKeyToAccount, generatePrivateKey } from 'viem/accounts';

// Generate Private Key and Account
const privateKey = generatePrivateKey();
const account = privateKeyToAccount(privateKey);

// Create Universal Signer
const universalSigner = {
  chain: CONSTANTS.Chain.EVM.sepolia.name, // Sepolia Testnet
  chainId: CONSTANTS.Chain.EVM.sepolia.chainId,
  account: account.address,
  signMessage: async (data: Uint8Array) => {
    const signature = await account.signMessage({
      message: { raw: data },
    });
    return hexToBytes(signature);
  },
};
```

### Initialize the SDK

Initialize the Push Chain SDK with the Universal Signer:

```typescript
const pushChain = await PushChain.initialize(universalSigner);
```

### Define Transaction Payload

```typescript
const email = {
  title: 'Hello old friend from Solana!',
  message: 'Greetings from Ethereum world.',
};
```

### Specify Recipients

```typescript
const recipients = [
  {
    chain: CONSTANTS.Chain.Solana.devnet.name, // Solana Devnet
    chainId: CONSTANTS.Chain.Solana.devnet.chainId,
    account: 'ySYrGNLLJSK9hvGGpoxg8TzWfRe8ftBtDSMECtx2eJR',
  },
];
```

### Send the Transaction

```typescript
const tx = await pushChain.tx.send(recipients, {
  category: 'example-category',
  data: new TextEncoder().encode(JSON.stringify(email)),
});
```
