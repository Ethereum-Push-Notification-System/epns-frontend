# Fetch Transaction Overview

The `Fetch Transactions` functionality in Push Chain allows developers to retrieve transaction details efficiently, leveraging the network's sharding and decentralized storage mechanisms. This page covers the setup and usage of transaction fetching with the Push Chain SDK.

To fetch transactions, we will use the `get` function from the `pushChain.tx` object.

## Get transaction function

The `pushChain.tx.get` function allows developers to fetch transactions by their hash, by sender address or recipient address.

## Get Function Parameters

```typescript
get: (
  reference?: UniversalAccount | string | '*' = '*',
  options?: {
    raw?: boolean;
    category?: string;
    startTime?: number;
    order?: Order;
    page?: number;
    limit?: number;
    filterMode?: 'both' | 'sender' | 'recipient';
  }
) => Promise<BlockResponse | SimplifiedBlockResponse>;
```

| Param       | Type               | Remarks                               | Default         |
|-------------|--------------------|---------------------------------------|-----------------|
| `reference`      | `UniversalAccount`, `string`, `'*'`       | Specifies the query target: `'*'` for all transactions, a transaction hash, or a UniversalAccount. | `*`                |
| `raw`            | `boolean`                                | If `true`, returns the raw `BlockResponse`. If `false`, returns a `SimplifiedBlockResponse`. For most cases use default `raw = false`.     | `false`            |
| `category`       | `string`                                 | Filters transactions by category (e.g., application-specific tags).                               | `undefined`        |
| `startTime`      | `number` (timestamp)                     | Fetches transactions starting from this timestamp.                                                | Current timestamp  |
| `order`          | `Order` (`'ASC'` or `'DESC'`)            | Determines the sort order of transactions (`'ASC'` for ascending, `'DESC'` for descending).        | `'DESC'`           |
| `page`           | `number`                                 | Specifies the page number for paginated results.                                                  | `1`                |
| `limit`          | `number`                                 | Sets the maximum number of transactions to fetch per page.                                        | `30`               |
| `filterMode`     | `'both'`, `'sender'`, `'recipient'`       | Determines the query type: `'both'` fetches all, `'sender'` fetches sent, `'recipient'` fetches received. | `'both'`           |

--------

Here below is an example of how to fetch a transaction by its hash.

```typescript
const transaction = await pushChain.tx.get('177482c5a504f3922875c216f71a2b236f344cfbf334f97c8f59547e1e21fb23');
```

## Examples

### Get Transaction by hash

Fetch a transaction that has the hash `177482c5a504f3922875c216f71a2b236f344cfbf334f97c8f59547e1e21fb23`.

```typescript
const transaction = await pushChain.tx.get('177482c5a504f3922875c216f71a2b236f344cfbf334f97c8f59547e1e21fb23');
```

### Get All Transactions By Category

```typescript
const transactionByCategory = await pushChain.tx.get('*', {
  category: 'CUSTOM:SAMPLE_TX',
});
```

### Get Transaction By Sender address

We will fetch transactions sent by this CAIP-10 address: `push:devnet:pushconsumer1l8wd6ucrwf43stuavxwfc9jmr5emlkr66guml6`.

```typescript
const transctionBySender = await pushChain.tx.get(
  {
    chain: CONSTANTS.Chain.Push.devnet.name,
    chainId: CONSTANTS.Chain.Push.devnet.chainId,
    account: 'pushconsumer1l8wd6ucrwf43stuavxwfc9jmr5emlkr66guml6',
  },
  { filterMode: 'sender' }
);
```

### Get Transaction By Receiver address

We will fetch transactions received by this CAIP-10 address: `push:devnet:pushconsumer1l8wd6ucrwf43stuavxwfc9jmr5emlkr66guml6`.

```typescript
const transctionBySender = await pushChain.tx.get(
  {
    chain: CONSTANTS.Chain.Push.devnet.name,
    chainId: CONSTANTS.Chain.Push.devnet.chainId,
    account: 'pushconsumer1l8wd6ucrwf43stuavxwfc9jmr5emlkr66guml6',
  },
  { filterMode: 'recipient' }
);
```
