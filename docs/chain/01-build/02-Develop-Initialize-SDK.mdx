## Initialize SDK

For production, use `CONSTANTS.ENV.PROD`

For development, use `CONSTANTS.ENV.STAGING`

### `PushNetwork.instance` function return type

When initializing the SDK, you will get an instance of the `PushNetwork` class.

```typescript
export declare class PushNetwork {
  block: Block;
  tx: Tx;
  wallet: Wallet;
}
```

Here is a breakdown for each of public fields of that class.
- **Block**: The Block class in the PushNetwork handles operations related to blockchain blocks. It provides methods to initialize a block instance, serialize and deserialize block data, and fetch blocks from the network.
- **Tx**: The Transaction class in the PushNetwork is designed to manage transactions within the network. It provides functionality to create unsigned transactions, serialize and deserialize transaction data, and send transactions to the network.
- **Wallet**: The Wallet class manages wallet interactions, including connecting to a wallet, signing data, and checking the connection status.
