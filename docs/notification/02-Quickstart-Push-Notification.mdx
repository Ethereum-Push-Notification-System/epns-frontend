### Initialize PushAPI Instance

<Tabs className="codetabs" groupId="code-examples">
  <TabItem value="js" attributes={{className: "codetab js"}} default>

```js
// Creating a random signer from a wallet, ideally this is the wallet you will connect
const signer = ethers.Wallet.createRandom()

// Initialize wallet user, pass 'prod' instead of 'staging' for mainnet apps
const userAlice = await PushAPI.initialize(signer, { env: 'staging' })
```

  </TabItem>

### Stream / Real time updates / Socket

<Tabs className="codetabs" groupId="code-examples">
  <TabItem value="js" attributes={{className: "codetab js"}} default>

```js
// To listen to real time notifications
userAlice.stream.on('STREAM.NOTIF', (data) => {
  console.log(data)
})
```

  </TabItem>
