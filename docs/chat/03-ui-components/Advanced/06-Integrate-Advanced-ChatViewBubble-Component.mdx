# ChatViewBubble

Simplify Development: Instantly Integrate Pre-built Chat Bubble UI for Your Dapp

ChatViewBubble serves as the fundamental building block for a chat view, allowing users to showcase individual chat messages while formatting them based on the provided payload. This essential component facilitates the presentation and customization of single chat messages with ease.

The ChatViewBubble is a powerful component that automatically embeds Push Frames and supports both the Open Frames Specification and Farcaster Frames Spec right out of the box. As a developer, if you're aiming to enhance your Chat Application without any additional effort, this component provides a fully frame-supported client straight out of the box.

### Customization Parameters

| Param                       | Type              | Default | Remarks                                  |
| --------------------------- | ----------------- | ------- | ---------------------------------------- |
| _`decryptedMessagePayload`_ | `IMessagePayload` | -       | It is a decrypted message payload object |

> Note: Parameters _`in this style`_ are mandatory.

> Note: Refer [ChatUIProvider](/docs/chat/ui-components/Advanced/ChatUIProvider/) for details on its paramters.
