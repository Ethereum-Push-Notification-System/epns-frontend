We wanted it to be easier for developers to add push chat in their dapp without having to worry about the UI or calling apis, that's why we built these web components which developers can import directly from the `@pushprotocol/uiweb` package and add push chat in their dapp just with few lines of code!

for example in the below image we have added the push chat view component in the [`push.org/brb`](https://push.org/brb) website

<ModalContainer>
  <img src={require("/static/img/usecases/chat.png").default} />
</ModalContainer>
in order to add the above chatview component the developer just need to write few lines of code
<Tabs className="codetabs" groupId="code-examples">
<TabItem value="js" attributes={{className: "codetab js"}} default>

```typescript
import { ChatViewComponent } from "@pushprotocol/uiweb";

export function Chat() {
  return (
    <ChatViewComponent
      chatId="b8e068e02fe12d7136bc2f24408835573f30c6fbf0b65ea26ab4c7055a2c85f1"
      limit={10}
      isConnected={true}
    />
  );
}
```
