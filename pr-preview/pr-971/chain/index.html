What is Push Chain?

Push Chain is a shared state blockchain for universal applications. It is designed to allow transactions from any chain (ETH, Solana, EVM, non-EVM), has 1s fast finality and has true scalability (read and write throughput increases as the number of nodes increases).It introduces some ground breaking innovations such as consumer txs (order not important), parallel validators and dynamic sharding and universal smart contracts. The result is a layer 1 with the ability to become a shared settlement layer for all L1s 🤝 L2s 🤝 L3.
